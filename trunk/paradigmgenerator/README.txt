This package is a tool for testing the inflection classes for SMOR. 
It requires a UTF-8 commandline environment and comes with its own binaries
for the SFST tools fst-infl2 and fst-compiler-utf8 on the platforms
Linux-i386 and OSX (i386, x86_64, ppc). In order to start just type

python paradigmtester.py

and follow the usage instructions.
